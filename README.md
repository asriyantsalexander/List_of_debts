# Обработка карточек задолженности

Веб-приложение для извлечения данных из PDF-карточек задолженности в таблицу Excel.

## Деплой на Railway

1. Создайте репозиторий на GitHub и загрузите все файлы из этой папки
2. В Railway: **New Project → Deploy from GitHub repo**
3. Выберите репозиторий — Railway автоматически подхватит Dockerfile
4. Приложение запустится, Railway выдаст публичный URL

## Локальный запуск (для тестирования)

```bash
pip install -r requirements.txt
python app.py
```

Откройте http://localhost:5000

## Использование

1. Откройте URL приложения
2. Загрузите архив (.rar / .zip / .7z) с PDF-карточками
3. Нажмите «Сформировать таблицу»
4. Скачайте готовый файл Excel
